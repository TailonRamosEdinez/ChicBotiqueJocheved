## 🚀 Como Usar o Site

### 1. **Abrir o Site**
```bash
# Duplo clique em index.html
# Ou abra em um navegador
```

### 2. **Navegar**
- Use o menu superior para navegar
- Clique em produtos para detalhes
- Use filtros para buscar

### 3. **Comprar**
1. Clique em "Ver Detalhes"
2. Selecione tamanho, cor e quantidade
3. Clique "Adicionar ao Carrinho"
4. Visualize o carrinho
5. Finalize a compra

### 4. **Funcionalidades Extras**
- Adicione favoritos (❤️)
- Use a busca (🔍)
- Entre em contato via formulário
