## 💡 Dicas Úteis

### Para Customizar Cores
Edite `styles.css` linhas 6-13:
```css
:root {
    --primary-color: #2c1810;      /* Cor principal */
    --secondary-color: #d4a574;    /* Cor secundária */
    --accent-color: #e8d4c4;       /* Cor acentuada */
}
```

### Para Adicionar Produtos
Edite `products.js` e adicione ao array:
```javascript
{
    id: 16,
    nome: "Novo Produto",
    categoria: "vestidos",
    preco: 199.90,
    // ... resto dos dados
}
```

## 🆘 Problemas Comuns

### O site não carrega
→ Certifique-se que todos os arquivos estão na mesma pasta

### Carrinho não persiste
→ Verifique se LocalStorage está habilitado (F12 > Application > LocalStorage)

### Estilos não aparecem
→ Refreshe a página (Ctrl+F5) para limpar cache

### JavaScript não funciona
→ Abra Console (F12) e procure por erros
